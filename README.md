# Formulario de Inscripción - Hackathon Banco BISA

Prueba técnica para la posición de Desarrollador Frontend. Esta aplicación es un formulario público de inscripción que permite a los usuarios postularse validando reglas de negocio estrictas y consultando sus repositorios públicos de GitHub.

## Tecnologías Utilizadas

- **Core:** React 18, TypeScript, Vite.
- **Estilos y UI:** Tailwind CSS, shadcn/ui (Radix UI).
- **Manejo de Formularios y Estado:** React Hook Form.
- **Validación de Esquemas:** Zod.
- **Arquitectura:** Feature-Driven Architecture (basado en principios de Clean Architecture / bulletproof-react).

## Pre-requisitos

Asegúrate de tener instalado en tu entorno local:
- [Node.js](https://nodejs.org/) (versión 18 o superior recomendada).
- npm, yarn o pnpm.
